설명: 테이블사용법

사용하는 태그

  • <table>

  • <tr>

  • <th>

  • <td>

사용하는 속성

  • bgcolor

  • colspan

  • rowspan

추가적인 설명

  • 테이블 속성들

    • table : 테이블 생성

    • tr : 행(row)을 정의

    • td : 열 을 정의

    • border : 표의 테두리를 출력(값이 클수록 테두리의 두께가 두꺼워진다.)

    • width : 표의 가로길이(너비)

    • height : 표의 세로 길이(높이)

    • align : 표의 위치 - left, center, right

    • bgcolor : 표의 배경색 설정

    • cellspacing : 셀과 설 사이의 간격을 설정 ( 경계선의 두께)

    • cellpadding : 셀의 테두리와 셀 안의 내용간의 간격을 설정

  • colspan속성은 열 합치기기능으로 td, th 태그 안에서만 사용할 수 있다.

  • colspan 속성값은 몇개의 열을 합치는가에 따라서 숫자를 지정한다. 예) <td colspan = 3> : 3개의 열을 합치라는 의미

  • rowspan 속성은 행(row) 합치기 기능으로 td, th 태그 안에서만 사용할 수 있다.

  • rowspan 속성값은 몇개의 행을 합치는가에 따라서 숫자를 지정한다. 예)<td rowspan=3>: 3개의 행을 합치라는 의미

설명: 회원관리폼 만들기에 필요한 요소들

사용하는 태그

  • <form>

  • <input>

  • <caption>

  • <select>

  • <textarea >

추가적인 설명

  • form태그의 속성들

    • method : 값을 전달하는 방식을 설정하는 속성(get or post)

    • action : 값이 전달될 파일명을 설정

  • input태그의 속성들

    • size : 입력창의 크기 설정

    • maxlength : 최대 입력 가능한 글자수 설정

    • autofocus : 자동 포커스 기능

    • id : 각 양식의 태그를 구분하기 위한 역할

    • name : 값을 전단하기 위한 변수 역할 --넘거가는 곳(웹프로그램)에 id라는 변수에 값을 넘김

    • required : 입력 양식에 대한 필수 입력 검사

  • select 태그는 콤보 박스를 생성해주는 태그로서 option태그로 여러 요소를 나열해서 원하는것을 채크하는 방식이다.

  • select속성

    • autofocus: 페이지가 로드 되었을 때 해당 목록으로 바로 포커스가 맞춰진다.

    • disabled: 화면에는 보이지만 사용할 수 없게 만든다.

    • multiple: 여러 값을 선택할 수 있도록 한다.

    • name: 이름을 붙인다.

    • size: 목록을 펼치기 전에 보여질 목록의 개수를 정해준다.

    • selected: 이 속성이 추가된 요소를 미리 채크된 상태로 시작한다.

    • value: 서블릿으로 넘어갈 진짜 값을 입력하는 속성이다.

  • caption 태그는 테이블의 캡션(caption, 테이블이나 사진, 삽화 등에 붙는 설명)을 정의할 때 사용합니다.

  • table 요소는 단 하나의 caption 요소만을 명시할 수 있으며, caption 태그는 언제나 table태그 바로 다음에 위치해야 합니다.

  • caption 요소의 콘텐츠는 테이블 바로 위쪽에 가운데 정렬되어 표시되지만, CSS의 text-align이나 caption-side 속성을 사용하여 캡션의 위치나 정렬 방법 등을 변경할 수 있습니다.
