설명: javascript기초

추가적인 설명

  • window객체는 브라우저 관련 최상위 내장객체로서 보통은 생략을 한다.

  • 문자를 출력할때 쌍따옴표와 외따옴표를 모두 사용할 수 있다.

  • 문장의 끝에는 세미콜론을 생략할 수 있다.

  • 출력문 안에 태그값을 넣을 수 있다.

  • writeln은 줄바꿈을 해주지 못한다.

  • 자바스크립트의 주석은 java와 동일하다.

  • 이클립스 source에 toggle comment로 주석처리가 된다.(ctrl+shift+c)

  • 여러가지 출력 방법이 존재한다.(first)

  • 외부 js파일을 불러올 수 있다.(call)

  • 선언자 없이 변수 선언하면 전역변수가 된다.

  • 변수명은 대소문자를 구분한다.

  • 변수명은 영문자,숫자,$,_로 만들 수 있다.

  • 변수명의 첫 글자는 숫자로 시작할 수 없다.

  • 예약어로 설정된 단어들은 변수명으로 사용할 수 없다.

  • var선언자는 변수선언, 변수선언 후 동시에 값할당, 한번에 여러개의 값 할당, 재할당이 가능하다.(var)

  • let선언자는 변수선언, 변수선언 후 동시에 값할당, 한번에 여러개의 값 할당은 가능하나 재할당은 불가능하다.(let)

  • const선언자는 변수선언, 변수선언 후 동시에 값할당, 한번에 여러개의 값 할당은 가능하나 변수 선언과 동시에 반드시 초기값, 즉 초기화를 해야하며 재할당은 불가능하다.(const)

  • const선언자는 주로 상수를 만들때 사용하는 선언자이다.

  • 변수 출력할시 변수를 넘겨줄때 ,또는 + 둘다 사용가능하다.(variable)

  • javascript에 자료형이 없는 것은 아니다.단지 변수선언때 사용하지 않을 뿐이다.(variable)

설명: 사용자정의함수

사용하는 함수

  • parseInt

  • Number

  • String

  • eval

  • isNaN

추가적인 설명

  • 자바스크립트에서 함수를 만들때는 function을 사용해서 만든다.

  • 자바스크립트의 함수명은 대.소문자를 구분한다.

  • 자바스크립트는 인터프리트방식의 언어로서 함수 호출과 함수정의가 서로 다른 스크립트에서 코딩될때 함수호출이 함수정의보다 먼저나오면 함수가 실행되지 않는다.(function01)

  • 함수 호출과 함수정의가 서로 같은 스크립트에서 코딩될때 함수호출이 함수정의보다 먼저나와도 실행된다.즉 함수정의의 위치 상관없이 호출가능하다.이를 호이스팅기능이라고 부른다.(function01)

  • parseInt는 문자형 데이터를 정수형 데이터로 변환해준다.(builtin)

  • parseFloatt는 문자형 데이터를 실수형 데이터로 변환해준다.(builtin)

  • Number는 문자형 데이터를 숫자형 데이터로 변환해준다.(builtin)

  • String는 숫자형 데이터를 문자형 데이터로 변환해준다.(builtin)

  • eval는 문자형 산술식을 연산 수행해준다.(builtin)

  • isNaN는 숫자가 아니면(문자가 포함되면) true값을 리턴하는 함수이다.(builtin)

설명: 내장객체

사용하는 함수

  • Date()

  • document.getElementById(id값)

  • setTimeout(시행할 명령,1/1000초)

  • clearTimeout()

  • location.reload()

추가적인 설명

  • Date메소드

    • getFullYear():년도를 구해온다.

    • getMonth():월을 구해오는데 0~11이여서 +1해야 원하는 월을 구해온다.

    • getDate():일을 구해온다.

    • getHours():시간을 구해온다.

    • getMinutes():분을 구해온다.

    • getSeconds():초를 구해온다.

    • getDay():요일에 맞는 숫자를 구해오는데 일요일부터 0이고 토요일이 6이다.원하는 요일을 출력하고싶으면 요일 배열을 만들어서 숫자에 맞게 출력시켜야한다.

  • document.getElementById를 이용해 원하는 태그의 id값을 통해서 태그를 불러와서 속성값을 바꿀수 있다.(Date02)
  • setTimeout(시행할 명령,1/1000초)은 1/1000초 마다 시행할 명령을 반복하는 함수이다.window객체가 제공한다.(Date02)
  • clearTimeout()함수는 setTimeout() 명령을 멈추게하는 함수이다.(Date02)
  • clearTimeout()함수로 원하는 setTimeout() 명령을 멈추게하려면 setTimeout()자체를 하나의 변수에 넣어서 clearTimeout()에 넣는다.(Date02)
  • setTimeout()자체를 하나의 변수에 넣을때 변수 앞에 선언자 없이 하여 전역변수로 선언했다.(Date02)
  • onLoad이벤트를 통해서 바로 함수를 실행할 수 있다.(Date02)

  • Math속성,메소드

    • Math.E:오일러상수

    • Math.PI: 파이.

    • Math.abs(): 절대값.

    • Math.max(),Math.min(): 최대값, 최소값.

    • Math.ceil(): 올림.

    • Math.round(): 반올림.

    • Math.floor(): 내림.

    • Math.random(): 난수.

    • Math.pow(밑,지수): 거듭제곱.

    • Math.sqrt(): 제곱근.

  • location.reload()는 페이지를 새로고침해준다.

  • String속성,메소드

    • length:길이를 구해온다

    • charAt() : 인덱스 번호의 문자를 추출

    • indexOf(찾는 문자,시작 인덱스) : 시작 인덱스 이후로 가장 먼저 나오는 문자의 인덱스 값을 구해온다.

    • lastIndexOf(찾는 문자): 찾는 문자를 뒤에서 부터 찾아와 그 인덱스 값을 리턴한다.

    • toUpperCase():대문자로 변환 해주는 함수.

    • toLowerCase():소문자로 변환 해주는 함수.

    • substring(start index, end index): start index에서 부터 end index-1인덱스 까지의 문자열을 추출한다.

    • substr(start index,추출할 문자열의 길이):start index에서 부터 원하는 길이의 문자열을 추출한다.

사용하는 함수

  • document.getElementbyId(아이디값)

  • document.querySelector(선택자)

  • document.getElementbyName(이름)

  • document.getElementbyClassName(클래스)

  • document.querySelector(선택자)

  • onclick

  • focus

  • blur

추가적인 설명

  • html에서 태그는 보통 element라고 불린다.

  • html은 반드시 1개의 root element로 시작해야한다.

  • document.getElementbyId(아이디값),document.querySelector(선택자)는 한개의 element만 구해온다

  • 스크립트 상에 함수 위에다가 전역변수로서 Id값으로 element를 선언하면 javascript의 인터프리터적인 성격 때문에 정상적으로 인식되지 않는다.(focus01)
