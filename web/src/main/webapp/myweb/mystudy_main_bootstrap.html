설명: Pagination 사용 방법

추가적인 설명

  • Pagination느 컨탠츠가 많아 페이지를 나누어야 할때 페이지 이동을 더욱 쉽게해주는 기능이다.

  • 더이상 페이지가 없을때는 disable을 통해서 클릭을 막아두고, active를 통해서 현재 페이지를 알려줄 수 있다.(page02)

  • 원하는 선택자를 골라서 Pagination의 위치를 조정할 수 있다.(page03)
