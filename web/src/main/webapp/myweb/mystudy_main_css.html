설명: css diplay 속성

사용하는 속성

  • display :none,block,inline,inline-block

추가적인 설명

  • none으로 설정하면 안에 값이 보이지 않음.

  • block으로 설정하면 세로로 정렬된다.

  • inline으로 설정하면 가로로 정렬된다.

  • inline-block인 기본적으로 가로로 설정되며 안에 객체값 세부설정 가능하다.

설명: css background 속성

사용하는 속성

  • background-image: url()

  • background-size: 너비 높이

  • background-repeat: repeat-x,repeat-y,no-repeat

  • background-attachment: fixed,scroll

  • background-position: 키워드/background-position:x축/background-position:x축 y축

추가적인 설명

  • 기본적으로 body태그에 사용한다.

  • 기본적으로는 background는 repeat된다.

  • 각각에 그림을 layer라고 부른다.

  • size를 한개의 값으로 설정하면 너비로 인식하고, size를 두개의 값으로 설정하면 처음은 너비로, 두번째는 높이로 인식한다

  • size를 %로 설정하면 창에 크기에 맞게 사이즈가 맞춰진다.

  • repeat-x는 x축으로 반복,repeat-y는 y축으로 반복한다.

  • no-repeat속성을 사용하면 반복하지 않음.

  • background-attachment속성에 fixed를 사용하면 배경이 스크롤 내려도 고정되고, scroll로 하면 그 위치에 고정되어 스크롤 되어도 따라오지 않음.

  • background-position의 키워드 값으로 center,top,bottom이 있다.

설명: 미디어 쿼리

사용하는 속성

  • @media

  • @media sceen,print

  • @media (max-width: )(min-width: )

  • @media screen and (orientation: portrait,landscape)

추가적인 설명

  • 메타 태그에 뷰포트 설정을 통해서 단말기에 따라서 화면 스크린 출력 방식을 다르게 할 수 있다.(at_media_rule)

  • @media를 사용하여 단말기 종류에 따라서 출력 방식을 다르게 할 수 있다.

  • 위에 at_media_rule에서는 스크린상에 보여줄때와 프린트 할 때 각각 배경생을 다르게 하여 보여준다.

  • 외부 css파일로도 미디어속성에 종류에 따라서 다르게 출력할 수 있다.(media_attribute)

  • 픽셀값을 통해서 단말기의 종류를 파악해서 화면 출력값을 다르게 할 수 있다.(media_feature)

  • 오리엔테이션을 통해서 단말장치에 맞게 출력을 가로,세로 각각 따로 설정할 수 있다.portrait은 세로,landscape은 가로로 출력한다. (orientation)

  • 픽셀 크기에 따른 단말기에 맞게 메뉴와 컨탠츠의 배치를 설정한다.스마프폰 일때는 float값을 해제해서 아래로 정렬된다.(pattern1)

  • pattern2는 pattern1과는 다르게 오른쪽에 메뉴가 있다.

  • pattern3,4는 pattern1,2와는 다르게 body배열순서를 컨탠츠를 먼저하고 메뉴를 나중에 해서 스마트폰 일때는 float값을 해제되어 있어서 컨탠츠가 먼저 출력된다.
