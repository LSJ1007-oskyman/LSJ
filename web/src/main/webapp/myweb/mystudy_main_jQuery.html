사용하는 함수

  • $('태그 또는 선택자').css('속성')

  • $('태그 또는 선택자').css('속성','변경할 값')

  • $('태그 또는 선택자','태그 또는 선택자').css('속성','변경할 값')

추가적인 설명

  • $('태그 또는 선택자').css('속성')은 해당 태그 또는 선택자의 속성 값을 가져온다.(css01)

  • $('태그 또는 선택자').css('속성','변경할 값')은 해당 태그 또는 선택자의 속성 값을 변경할 값으로 한다.(css01)

  • $('태그 또는 선택자').css('속성','변경할 값')안에 {}를 넣어서(.css({속성: 변경할 값,속성: 변경할 값...}))여러개를 한번에 변경가능하다.

  • $('태그 또는 선택자','태그 또는 선택자').css('속성','변경할 값')를 통해서 $안에 있는 태그 또는 선택자 모두가 변경된다.(css06)

사용하는 함수

  • $('태그 또는 선택자').dblclick(function(){})

  • $('태그 또는 선택자').attr('속성')

  • $('태그 또는 선택자').attr('속성','변경할 값')

  • eq

  • this

  • $('태그 또는 선택자').mouseover(function(){})

  • $('태그 또는 선택자').mouseout(function(){})

  • $('태그 또는 선택자').keyup(function(){})

추가적인 설명

  • $('태그 또는 선택자').click(function(){})를 통해서 해당 태그 또는 선택자의 값의 클릭 이벤트 여부를 알 수 있다.(click01)

  • $('태그 또는 선택자').attr('속성')를 통해서 태그 또는 선택자 내의 속성값을 가져올 수 있다.(click01)

  • $('태그 또는 선택자').attr('속성','변경할 값')를 통해서 태그 또는 선택자 내의 속성값을 변경할 수 있다.(click01)

  • $('태그 또는 선택자').attr('속성','변경할 값')는 .attr('속성','변경할 값')를 이어붙여서 한번에 여러개를 변경할 수 있다.(click01)

  • <a>태그도 이벤트객체로 할 수 있다.(click02)

  • <a>태그에 하이퍼링크값이 존재할 경우 return false;를 $('태그 또는 선택자').click(function(){})안에 넣어서 동작을 막을 수 있다.(click03)

  • eq는 가져오는 태그 또는 선택자의 객체가 여러개인 경우 eq(번호)를 통해서 순서대로 객체를 불러올 수 있게 도와준다.(click04)

  • this는 이벤트를 발생시킨 객체를 의미하며 굳이 eq를 사용하여 하나하나 불러올 필요가 없다.(click05)

  • $('태그 또는 선택자').dblclick(function(){})를 통해서 해당 태그 또는 선택자의 값의 더블클릭 이벤트 여부를 확인 할 수 있다.(dblclick01)

  • $('태그 또는 선택자').mouseover(function(){})를 통해서 해당 태그 또는 선택자의 값의 마우스가 올라갔는지 알 수 있다.(mouseover01)

  • $('태그 또는 선택자').mouseout(function(){})를 통해서 해당 태그 또는 선택자의 값의 마우스가 내려갔는지 알 수 있다.(mouseover01)

  • $('태그 또는 선택자').mouseover(function(){}).mouseout(function(){})로 붙여서 한번에 정의할 수 있다.(mouseover01)

  • $('태그 또는 선택자').val()은 입력양식(input),선택양식(select)의 입력,선택값을 구해올때 사용된다.(keyup01)

  • $('태그 또는 선택자').keyup(function(){})는 해당 태그 또는 선택자의 값의 keyup이벤트를 관리한다.(keyup01)

  • 값의 길이를 구하는 함수가 따로 없기 때문에 length속성을 사용한다.(keyup01)

  • $('태그 또는 선택자').change(function(){})는 선택양식의 값의 변화가 있는지를 알 수 있게 해준다.(change01)

  • $('태그 또는 선택자').removeAttr('속성')은 해당 속성의 적용을 해제시킨다.(change01)

  • $('태그 또는 선택자').removeAttr('readonly')를 하는 이유는 정해진 도메인을 선택했을 때 readonly가 되어서 다시 직접입력으로 돌아와도 남아있기 때문이다.(change01)

  • $('태그 또는 선택자').submit(function(){})는 태그 또는 선택자의 submit이벤트를 관리한다.(submit01)

  • 유효성 검사를 위해서 값이 넘어가기 전에 처리하기 위해서는 $('태그 또는 선택자').submit(function(){})의 태그 또는 선택자를 submit버튼이 아닌 반드시 form으로 해야 값이 넘어가지 않는다.

  • $('태그 또는 선택자').on('이벤트이름',function(){})는 원하는 이밴트를 on을 통해서 보다 간편하게 처리할수 있게 해준다.

설명: form객체들

사용하는 함수

  • $('태그 또는 선택자').val()

  • $('태그 또는 선택자').text()

  • $('태그 또는 선택자').prop('속성')

  • $('태그 또는 선택자').prop('속성','변경값')

  • $('태그 또는 선택자').removeProp('속성','변경값')

  • $('태그 또는 선택자').is('가상선택자')

추가적인 설명

  • $('태그 또는 선택자').val()은 입력양식(input),선택양식(select)의 입력,선택값을 구해올때 사용된다.(form01)

  • 입력양식(input),선택양식(select)의 값을 설정할 때도 $('태그 또는 선택자').val('입력값')을 사용한다.(form01)

  • $('태그 또는 선택자').text()는 태그 또는 선택자의 입력된 값을 가져온다.(text01)

  • :checkbox는 가상 선택자로서 모든 checkbox로 태그된 모든 요소를 가져온다.(checkbox01)

  • prop()는 속성(property)에 속성값을 설정하거나, 속성으로 속성값을 구해오는 역할을 하는 메소드이다.(checkbox01)

  • radio버튼은 반드시 name값을 같이 설정해서 묶어야 다중선택을 방지할 수 있다.(radio01)

  • is()는 radio버튼이나 checkbox의 선택 여부를 판별할때 사용되는 메소드 선택되면 true, 선택되지 않으면 false를 리턴한다.(radio01)

  • :radio는 가상 선택자로서 모든 radio로 태그된 모든 요소를 가져온다.(radio02)

  • select는 change로 이벤트 캐치를 한다.(select01)

  • 정규 표현식(Regular Expression) 정규 표현식 특정한 규칙을 가진 문자열의 집합을 표현하는 데 사용하는 형식 언어이다.(regular01)

  • ^:정규식 시작,+:1회이상, []:형식 지정,$:정규식 종료.(regular01)

설명: 여러함수들

사용하는 함수

  • $.each(list,function(index,value){})

  • $('태그 또는 선택자').append('붙일값')

  • $('태그 또는 선택자').hide()

  • $('태그 또는 선택자').show()

  • $('태그 또는 선택자').text()

  • $('태그 또는 선택자').html()

  • $('태그 또는 선택자').toggle()

추가적인 설명

  • $.each(list,function(index,value){})는 배열요소들의 처리를 위한 기본 예제로서 함수 안에서 index와 value를 이용해서 원하는 과정을 처리한다.(each01)

  • $('태그 또는 선택자').append('붙일값')는 태그 또는 선택자뒤에 html코드상으로 붙일값을 적용한다.(each01)

  • array = [{name:'naver',link:'http://www.naver.com'}]<-이런식으로 키값과 저장된 값으로 원소를 지정할 수 있다.(each02)

  • $('태그 또는 선택자').hide()는 태그 또는 선택자를 웹상에서 감춘다.(hide01)

  • $('태그 또는 선택자').show()는 태그 또는 선택자를 웹상에 나타나게 한다.(hide02)

  • jQuery상에서는 버튼뿐만아니라 <p>태그도 클릭 이벤트 설정이 가능하다.(hide03)

  • $('태그 또는 선택자').text()는 모든 태그 또는 선택자의 내용을 읽어온다.(text_html01)

  • $('태그 또는 선택자').text()는 <>를 문자로 인식해 그대로 출력한다.(text_html02)

  • $('태그 또는 선택자').html()첫번째 태그 또는 선택자의 내용만 읽어온다.(text_html01)

  • $('태그 또는 선택자').html()는 <>를 태그로 인식해 태그 처리한다.(text_html02)

  • $('태그 또는 선택자').toggle()은 선택한 요소가 보이면 보이지 않게, 보이지 않으면 보이게 합니다.(toggle01)

설명: jquery를 이용한 ajax

사용하는 함수

  • $('태그 또는 선택자').load('파일')

  • window.open("팝업창에 실행될 문서명","윈도우 이름","옵션"))

  • $.ajax({
    url : "전송 페이지",
    type : "전송 방식(get, post방식),
    data : "전송할 데이터",
    dataType : "요청한 데이터타입("html","xml","json","text"),
    success : function(result){
    콜백함수로 돌려 받은 후 실행될 문장;
    }
    });

  • $.post( "요청할 파일명",
    { “key” : value } ,
    function(msg){
    콜백함수로 돌려 받은 후 실행될 문장;
    }
    );

  • $.get(
    "요청할 파일명",
    function(data){
    콜백함수로 돌려 받은 후 실행될 문장;
    }
    );

  • $.getJSON(
    "요청할 json파일명",
    function(data, textStatus){
    콜백함수로 돌려 받은 후 실행될 문장;
    }
    );

추가적인 설명

  • .load()는 해당 파일의 내용을 태그 또는 선택자에 출력하는 함수이다.(load01)

  • .load()에 파일자리에 'menu.html li'와 같이 후손 선택자 형식이 와도 정상적으로 실행된다.(load03)

  • $.ajax()에 data는 전송할 데이터가 있을때, datatype은 받을 데이터가 있을때만 사용한다.(ajax01)

  • xml파일은 반드시 1개의 루트만 가진다.(get01)

  • get01 jquery내의 코드 흐름은 다음과 같다. 태그값을 테이블의 제일 위에 두고 item값을 찾는데, 여러개이므로 each통해서 각각 함수를 실행하며 item태그 안에 있는 속성인 id과 name을 attr을 통해서 찾아서 출력, item태그 하위 태그인 price와 description을 find함수를 찾아서 출력한다.(get01)

  • getjson콜백함수의 매개변수 textStatus는 불러오기를 성공했으면 success보낸다.(getjson01)

  • ajax03은 getjson함수를 ajax로 불러온 것이다.(ajax03)
